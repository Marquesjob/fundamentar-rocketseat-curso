----------------------------------------------------------------------------------------------

## Basic font properties

* font-family
* font-weight
* font-style
* font-size


----------------------------------------------------------------------------------------------

## Font Family 

* Tipo de fonte de um elemento
* Lista de fontes e ordem de prioridade
* inclui *fallback* font

```css
p {
    font-family: "Times New Roman", Times, serif;
}
```
    -   serif
    -   sans

-----------------------------------------------------------------------------------------------

## Font Weight

* Peso da fonte

```css
p{
    font-weight: bold;
}
```
-----------------------------------------------------------------------------------------------

## Font style

* O estilo da fonte

```css
p {
    font-style: italic;
}
```
------------------------------------------------------------------------------------------------

## Font Size 

*  O tamanho da fonte

```css
p {
    font-size: 18px;
}
```
